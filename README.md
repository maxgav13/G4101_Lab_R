# Laboratorios Geología Numérica G-4101

Este repositorio contiene los materiales de laboratorio del curso **Geología Numérica G-4101**, impartido en la [Escuela Centroamericana de Geología](https://www.geologia.ucr.ac.cr/) de la [Universidad de Costa Rica](https://www.ucr.ac.cr/).

## Descripción

El curso introduce el uso de herramientas computacionales y métodos cuantitativos aplicados a las geociencias. Los laboratorios están desarrollados en [Quarto](https://quarto.org/) (`.qmd`) y utilizan **R** como lenguaje de programación principal.

## Contenido

### Introducción

- `G4101_Intro_R.qmd` — Introducción al lenguaje R
- `G4101_Intro_Quarto.qmd` — Introducción a Quarto

## Tecnologías

- **R** ≥ 4.0
- **Quarto** para la generación de documentos reproducibles (HTML, PDF, DOCX)
- Paquetes del ecosistema **tidyverse** y otros especializados en geociencias
